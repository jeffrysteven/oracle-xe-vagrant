### How can I set it up? ###

* Requirements:
  - Virtual Box 5.1
  - Vagrant 1.9
* Getting started:
  - Clone this repo
  - Run `vagrant up` to start and provision

### Connect database with following setting: ###

hostname: localhost
port: 49161
sid: xe
username: system
password: oracle
(Password for SYS & SYSTEM)

-- Login by SSH --

ssh root@localhost -p 49160
password: admin
